Adds `Content-Security-Policy: upgrade-insecure-requests` header if on HTTPS. Redirects to HTTPS if not on HTTPS and the browser sends `HTTP_UPGRADE_INSECURE_REQUESTS` header.

== Description ==

* Adds `Content-Security-Policy: upgrade-insecure-requests` header if on HTTPS
* Redirects to HTTPS if not on HTTPS and the browser sends `HTTP_UPGRADE_INSECURE_REQUESTS` header.

This is a simple, no-config plugin with focus on short, high quality code.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done! The plugin does not have a configuration interface.
